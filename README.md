# 📝 To-Do App (JavaScript Vanilla)

Aplicação web simples de lista de tarefas desenvolvida com **HTML, CSS e JavaScript puro**, focada em manipulação do DOM, armazenamento local e boas práticas de organização de código.

## 🚀 Funcionalidades

* Adicionar tarefas
* Marcar como concluída
* Filtrar por:

  * Todas
  * Pendentes
  * Concluídas
* Editar tarefas
* Excluir tarefas
* Limpar tarefas concluídas
* Contador automático
* Armazenamento no LocalStorage
* Interface moderna e responsiva

## 💻 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

* Manipulação do DOM
* Eventos do JavaScript
* Estruturação de lógica frontend
* Persistência de dados no navegador
* Organização de código em funções reutilizáveis

## 📌 Possíveis melhorias futuras

* Drag and Drop de tarefas
* Dark mode
* Backend com API
* Login de usuário
* Deploy online
